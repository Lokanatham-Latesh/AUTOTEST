# 📝 **Overview**

**AUTOTEST Web App** is a **full-stack AI-powered web testing automation platform** that:

- Crawls websites recursively using BFS
- Extracts page metadata (forms, inputs, UI components, links)
- Uses **LLMs (Large Language Models)** to generate:

  - Test Scenarios
  - Test Cases
  - Python Selenium scripts

- Stores results, logs, settings, and analysis data
- Provides a modern dashboard to manage all testing workflows

The application **removes the need for manual test writing** by leveraging Generative AI and structured metadata.

  - Done

### 📄 **3. Page Management**

- View all extracted pages
- Page status tracking
- Open Page Info to view:

  - Extracted metadata
  - DOM structure

### 🧪 **4. Test Scenario Management**

- Auto-generated by LLM
- Manual scenario creation supported
- Each scenario includes:

  - Title
  - Test metadata (JSON)
  - Category (functional / auth / validation / UI / navigation…)
  - Python Selenium test script
  - Script path

# ⚙️ **Installation & Setup**

## 🔧 Prerequisites

- Python 3.10+
- Node.js 18+
- MySQL 8+
- npm or yarn
- API keys for LLM providers (OpenAI / Gemini / etc.)

---

## 🐍 **Backend Setup (FastAPI + MySQL)**

```bash
git clone <repo-url>
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### Set up `.env` file:

```
DATABASE_URL=mysql+pymysql://root:password@localhost/autotest
JWT_SECRET="your-secret-key"
OPENAI_API_KEY="your-api-key"
```

Run backend:

```bash
uvicorn app.main:app --reload
```

---

## 🎨 **Frontend Setup (React + Tailwind + shadcn-ui)**

```bash
cd frontend
npm install
npm run dev
```

Frontend is available at:

👉 `http://localhost:5173`

# 🚀 **Usage**

### 1. Login / Register

- Login using credentials
- Default newly registered user → **User role**

### 2. Add a Site

- Provide site title & base URL
- Site record will be created

### 3. Start Parsing Pages

- Backend crawls pages via BFS
- Metadata extraction performed

### 4. View Pages

- Inspect extracted pages
- Open **Page Info**

### 5. Generate Test Scenarios

- LLM generates:

  - Scenarios
  - Test cases
  - Python Selenium scripts

### 6. Manage Settings (Admin Only)

- Admin configures all system-level parameters that control how the AUTOTEST Web Application functions.
